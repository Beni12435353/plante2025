📤 Pas cu Pas: Publicare pe GitHub Pages

1. Creează un nou repository pe GitHub

  1. Mergi pe github.com

  2. Click pe "+" sus-dreapta → "New repository"

  3. Completează:

  3. Click Commit changes jos.

3. Activează GitHub Pages

  1. Mergi în tabul Settings al repo-ului

  2. În meniu lateral, click pe "Pages" (sau „Pages & Deployment”)

  3. La „Source”, alege:

    • Branch: main

    • Folder: / (root)

  4. Click pe Save
